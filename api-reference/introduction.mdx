Once your container definitions and API keys have been set up, they can then be accessed using the required endpoints detailed in the API reference.

## Getting Started

In order to access your container data you must:

- Create an API key within the SlyceIO UI for the required container definition 
- Use the API key to retrieve an encrypted bearer token via the [authentication token endpoint](/api-reference/endpoint/authentication)
- Add the bearer token to the authorization header for any of the [document endpoints](/api-reference/endpoint/create) you wish to use

<Note>
Bearer token security is used for all <b>Document Endpoints</b>.
</Note>
