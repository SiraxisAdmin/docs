---
title: 'Postman Template'
description: 'Project template used to access your container data via Postman'
---

<Info>
  - Copy the JSON template into Postman.
  - Obtain a bearer token from the authentication token endpoint by using the relevant API key in the `SlyceIO-Api-Key` header.
  - Replace the `containerDefinitionId` and `documentId` path variables with the required values.
  - Ensure all `partitionKeys` query parameter values are **url encoded**.
</Info>
