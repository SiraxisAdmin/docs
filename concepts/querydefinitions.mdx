## Overview

Query definitions allow you to define multiple queries for the selected container definition using Azure Cosmos DB SQL syntax. 

Both **static** and **parameterized** queries are available. Parameterized queries use the following syntax `@{{myparametername}}` for declaring query parameters.
```SQL Sample query
 select * from c 
 where c.id = '@{{idParameter}}' 
 and c.version = 1
```

<Tip>
Queries can be <b>tested</b> to ensure they function as expected before being made available through the API.
</Tip>
