## Overview

Creating a connection is your starting point and defines the **Azure Cosmos DB account endpoint and key** that is used for managing container definitions and associated query definitions within SlyceIO.

<Note>
<b>Read/Write</b> Azure Cosmos DB account keys are required for <b>inserting, updating, deleting or querying</b> container data.
</Note>
<Note>
<b>Readonly</b> Azure Cosmos DB account keys are sufficient if you just want to <b>query</b> container data.
</Note>
<Note>
Azure Cosmos DB account endpoints must be <b>publicly</b> accessible.
</Note>

## Context Menu
You can use the context menu to **edit**, **delete** or **test** your existing connections.

<img
  style={{ borderRadius: '0.5rem' }}

<Warning>
Connection endpoints cannot be <b>changed</b>.
</Warning>
<Warning>
Connections cannot be <b>deleted</b> when associated with one or more container definitions.
</Warning>
