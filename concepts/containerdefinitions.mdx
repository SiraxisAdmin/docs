## Using Container Definitions

Container Definitions allow you to store container metadata for the required connection and database, with the option to add a **JSON schema** to ensure data integrity when inserting or updating data for the specified container.

## Context Menu
Once created you can use the context menu to edit or delete existing container definitions, manage [query definitions](/concepts/querydefinitions), view the container API specification or any associated API keys.
