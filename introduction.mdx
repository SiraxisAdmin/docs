## Quickstart

Learn how to start accessing your Azure Cosmos DB container data in less than 5 minutes.

<Steps>
  <Step title="Create an account">
    Creating an account only requires a valid email address. A time dependant security code will be sent to your email address and then used to log in to the UI.
  </Step>
  <Step title="Set up your containers">    
    Create connections, container definitions, and query definitions. Secure container access with flexible API keys.
  </Step>
  <Step title="Access your data">
   Container data is accessed using the [SlyceIO data gateway](/api-reference/introduction). It contains a comprehensive set of instructions and interactive code for communicating with the gateway.
  </Step>
</Steps>
